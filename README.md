# rtc-locksmith-refactor

Набор артефактов стека «робот‑слесарь» (скрипты Lua, проекты Tesla SCADA и aubope) для систематического рефакторинга, упорядочивания архитектуры, повышения тестопригодности и безопасности работы робота. [web:87][web:129]

## Цели проекта

- Структурировать существующий код и проекты управления роботом‑слесарем.
- Выделить уровни архитектуры (управление, визуализация, процедуры, окружение).
- Ввести спецификации операций слесаря и связать их с кодом и тестами.
- Обеспечить воспроизводимое окружение для дальнейшей разработки и сопровождения.

## Структура репозитория

```text
lua-runtime/        # Скрипты Lua робота, тесты и утилиты
scada-tesla/        # Проект Tesla SCADA (экраны, теги, логика)
aubope/             # Проект и процедуры aubope, файлы окружения
docs/               # Документация и спецификации операций
tooling/            # Служебные скрипты, хуки git и пр.
